## Period
アスリートとトレーナーが繋がるアプリ

### 技術スタック

- **Framework**: Next.js 16 (App Router)
- **Database**: PostgreSQL + Prisma
- **Authentication**: Better Auth
- **Realtime**: Pusher (WebSocket alternative)
- **Styling**: CSS Modules
- **Deployment**: Vercel / Render.com

### 主な機能

- アスリート/トレーナーのロール管理
- パートナー連携システム
- **リアルタイムチャット**（Pusher使用）
- ワークアウト管理
- カレンダー機能
